## Keyword-strategi (iOS App Store)

### Princip
Apple indexerer: **App Name** (30 chars) + **Subtitle** (30 chars) + **Keyword field** (100 chars).
Gentag IKKE ord på tværs af felter — Apple kombinerer dem automatisk.

### Primære keywords (høj volume)
```
photo cleaner, clean photos, delete photos, camera roll,
storage, free space, swipe, organize photos
```

### Sekundære keywords (medium volume, lav konkurrence)
```
photo sort, declutter, cleanup gallery, tidy photos,
photo manager, clear storage, swipe delete
```

### Long-tail keywords (lav volume, høj konvertering)
```
swipe left delete photo, tinder for photos, photo challenge,
challenge friends photos, sort camera roll fast
```

## Screenshot-strategi

Apple viser de første 3 screenshots i søgeresultater. De skal **sælge** på under 2 sekunder.

| # | Tekst (stor, fed) | Fokus | Konverteringsmål |
|---|-------------------|-------|------------------|
| 1 | "Ryd op på 1 minut" | Swipe-kort med billede | Hook: det er hurtigt |
| 2 | "Swipe venstre = slet" | Mid-swipe overlay | Forståelse: sådan virker det |
| 3 | "Udfordr dine venner" | Challenge resultat | Social proof / sjovt |
| 4 | "Sortér i mapper" | Settings med album | Premium feature peek |
| 5 | "Se din fremgang" | Stats skærm | Gamification |
| 6 | "Intet slettes uden dig" | Trash med restore | Trust / tryghed |

### A/B test-plan (App Store Connect)
- **Test A**: Screenshots med gradient-baggrund + device mockup
- **Test B**: Edge-to-edge screenshots uden mockup (større billede)
- **Test C**: Første screenshot = Challenge score (social hook)

## Review-strategi

- **Trigger in-app review prompt** efter:
  - Første 20 sorterede billeder ("du er i gang!")
  - Første challenge gennemført
  - 100 billeder milestone
- **Aldrig** prompt under swipe-flow (irriterer)
- Svar på ALLE reviews — Apple belønner aktive udviklere
